          # 1.0.0

          这是主题发布的第一个版本。

          布局组件结构：

          • Sidebar 目录侧边栏（默认主题组件）
          • Header 顶部
            • Aside 左侧边栏，包括头像、昵称、描述等
              • Nav 导航栏
              • Sidebar 移动端展示目录（默认主题组件）
                • Contact 联系方式，插入到 Sidebar 插槽中
            • SearchBox 搜索功能（默认主题组件）
          • Layout 布局组件，用以设置 md 的展示组件，以下是相互替换的组件
            • Blog 首页文章列表
              • Pagination 分页
            • Post 文章展示
              • Content 默认 md 转换的内容组件（默认主题组件）
              • Quote 格言（全局组件）
              • Vssue 评论
            • Tags 标签云展示
            • categories 分类云展示
            • Tag 单个标签的文章列表展示
              • Pagination 分页
            • Category 单个分类的文章列表展示
              • Pagination 分页
          • Footer 底部
