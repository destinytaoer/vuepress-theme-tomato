  • VuePress 入门

    • 2019-02-21
    • 技术文档

    VuePress 是一款使用 Vue 驱动的静态网站生成器，是 Vue 的作者 Evan You 为了方便文档的编写而开发的。

    • 默认主题与 Vue 官方文档一致
    • 简洁，少配置，高性能
    • Markdown 专为技术文档提供拓展
    • 自带 PWA
    • 自定义主题，可定制程度完全由自己决定

    官方文档：由于 1.x 还处于测试阶段，可能随时变更，所以建议选用 0.x 版本
